### Creating Operators
```python
# Create operator in current component
op('container1').create(topType, 'myTop')        # Create a TOP
op('container1').create(chopType, 'myChop')       # Create a CHOP
op('container1').create(sopType, 'mySop')         # Create a SOP
op('container1').create(datType, 'myDat')         # Create a DAT
op('container1').create(matType, 'myMat')         # Create a MAT
op('container1').create(compType, 'myComp')       # Create a COMP

# Create in root
root = op('/')
newOp = root.create(noiseTOP, 'noise1')
```

### Operator Connections
```python
# Connect output of op1 to input of op2
op('noise1').outputConnectors[0].connect(op('feedback1').inputConnectors[0])

# Or use shorthand
op('op1') >> op('op2')  # Not always available, use connectors for reliability
```

### Common Patterns

#### Audio Reactive Setup
```python
# Audio input -> Spectrum -> Analyze -> Drive visuals
audioIn = parent().create(audiodevinCHOP, 'audioIn')
spectrum = parent().create(audiospectrumCHOP, 'spectrum')
analyze = parent().create(analyzeCHOP, 'analyze')
spectrum.inputConnectors[0].connect(audioIn.outputConnectors[0])
analyze.inputConnectors[0].connect(spectrum.outputConnectors[0])
```

#### Feedback Loop
```python
noise = parent().create(noiseTOP, 'noise1')
feedback = parent().create(feedbackTOP, 'feedback1')
comp = parent().create(compositeTOP, 'comp1')
level = parent().create(levelTOP, 'level1')
transform = parent().create(transformTOP, 'transform1')

noise.outputConnectors[0].connect(comp.inputConnectors[0])
feedback.outputConnectors[0].connect(transform.inputConnectors[0])
transform.outputConnectors[0].connect(comp.inputConnectors[1])
comp.outputConnectors[0].connect(level.inputConnectors[0])
level.outputConnectors[0].connect(feedback.inputConnectors[0])
```

#### Instancing
```python
geo = parent().create(geometryCOMP, 'geo1')
geo.par.instanceop = op('grid1').path
geo.par.instancetx = 'tx'
geo.par.instancety = 'ty'
geo.par.instancetz = 'tz'
```
