## TOUCHDESIGNER PYTHON API REFERENCE

### Creating Operators
```python
# Create operator in parent component
op_name = parent().create(noiseTOP, 'my_noise')

# WRONG examples (DO NOT USE):
# parent().create(audiodevinCHOP, 'audio')     # ❌ Wrong: use audiodeviceinCHOP
# parent().create(hsvAdjustTOP, 'hsv')          # ❌ Wrong: use hsvadjustTOP
# parent().create(pointSpriteMAT, 'mat')        # ❌ Wrong: use pointspriteMAT
# parent().create(videoDeviceInTOP, 'cam')       # ❌ Wrong: use videodeviceinTOP

# CORRECT examples:
# parent().create(audiodeviceinCHOP, 'audio')   # ✅ Correct
# parent().create(hsvadjustTOP, 'hsv')           # ✅ Correct
# parent().create(pointspriteMAT, 'mat')         # ✅ Correct
# parent().create(videodeviceinTOP, 'cam')       # ✅ Correct
```

### Operator Connections
```python
# Connect output of op1 to input of op2
op('noise1').outputConnectors[0].connect(op('feedback1').inputConnectors[0])
```

### Setting Parameters
```python
op('noise1').par.resolutionw = 1920
op('noise1').par.resolutionh = 1080
op('noise1').par.monochrome = True
op('noise1').par.seed = 1
op('noise1').par.period = 2.0

# For menu parameters
op('composite1').par.operand = 'over'
```

### Expressions and References
```python
# Reference another parameter
op('transform1').par.tx.expr = "op('noise_chop').chan('chan1')"

# Time-based expressions
op('noise1').par.t.expr = "absTime.seconds * 0.1"
op('transform1').par.r.expr = "absTime.frame"
```

### Layout and Organization
```python
# Set operator position in network editor
op('noise1').nodeX = 100
op('noise1').nodeY = -200

# Set viewer and display flags
op('noise1').viewer = True
op('null1').display = True

# Set color
op('noise1').color = (0.5, 0.2, 0.8)

# Add comment
op('noise1').comment = 'Main noise source'
```

### Common Patterns

#### Audio Reactive Setup
```python
audioIn = parent().create(audiodeviceinCHOP, 'audioIn')
spectrum = parent().create(audiospectrumCHOP, 'spectrum')
analyze = parent().create(analyzeCHOP, 'analyze')
audioIn.outputConnectors[0].connect(spectrum.inputConnectors[0])
spectrum.outputConnectors[0].connect(analyze.inputConnectors[0])
```

#### Feedback Loop
```python
noise = parent().create(noiseTOP, 'noise1')
feedback = parent().create(feedbackTOP, 'feedback1')
comp = parent().create(compositeTOP, 'comp1')
level = parent().create(levelTOP, 'level1')
transform = parent().create(transformTOP, 'transform1')

noise.outputConnectors[0].connect(comp.inputConnectors[0])
feedback.outputConnectors[0].connect(transform.inputConnectors[0])
transform.outputConnectors[0].connect(comp.inputConnectors[1])
comp.outputConnectors[0].connect(level.inputConnectors[0])
level.outputConnectors[0].connect(feedback.inputConnectors[0])
```

#### Cleanup Existing Ops
```python
for c in parent().children:
    if c.name != me.name:
        c.destroy()
```
